Introduction to Operating Systems

Main Components of OS:

1. Kernel (core of OS)
  Controls CPU, Memory, Devices
2. System Calls
  Interface between OS
3. File System
  Organize Data (files/folders)
4. Device Drivers
  Communicate with hardware
5. User Interface
  CLI(Terminal) or GUI (Windows/Mac OS)



Tasks of an OS

Major responsibilites : (Process Management, Memory Management, File Management, Device Management, Security)


Types Of OS -(linux, windows, mac)

 Features
 • Open Source (free and customizable)
 • Strong community support
 • Extremely lightweight and powerful
 • Excellent for servers and development
 Popular Distros:
 • Ubuntu
 • Kali Linux
 • Debian
 • Fedora
 Use Cases
 ➢ Cloud
 ➢ DevOps
 ➢ Web servers (90% + Interent runs on linux)


What is distribution?
  A distribution = Linux Kernel + software + tools + Interface
  Linux is a Unix-like OS kernel combined with the tools -> become a full OS(distribution)


#UNIX: 
  -> One of the oldest operating system
  -> Developed at Bell Labs(1970s)
  -> Designed for multiuser + multitasking system
 Features
 • Strong security and permission models
 • Highly stable and reliable
 • Written mostly in C language
 • Designed for servers and enterprise systems
 Examples
 • IBM AIX
 • HP-UX
 • Solaris
 Use Cases:
 • Banking systems
 • Telecom Infrastructure
 • Large enterprise servers
 UNIX as a parent of Linux and macOS


# LINUX vs WINDOWS

 When to use Windows?
  -> You want ease of use
  -> Regular office work or gaming
  -> Need specific software (Ms office, Adobe)
  Windows = Ease + Compatibility + Desktop
 When to use LINUX?
  -> You want performance + Controls
  -> You like customization
  -> You are into programming/Devops/Cloud
  Linux = power + control + servers


====>Why Linux is Lightweight ?
       Minimal background process, runs only essential services,no unnecessary apps by default as as result uses less RAM,CPU and storage

====>How an organization decides? (linux or windows)

  1 Start with the Use Case
      Choose Linux when
        • Running Web servers, APIs, microservices
        • Using cloud platform
        • Need high performance + automation
      Choose Windows when
        • Use microsoft ecosystem
        • Need tools like
            o Active directory
            o NET applications
            o Microsoft office
  2 Cost Consideration
  3 Security Requirements
  4 Application Compatibility
  5 Team Skills
  6 Cloud and DevOps Strategy
  7 Enterprise Ecosystem


====> Can we run linux inside windows ?
    YES!
    1. WSL (Windows Subsystem for Linux) : Which runs linux inside windows
    2. VM (Virtual Machine) : Tools like Virtual Box, VMWare <Linuz runs as a seperate OS inside windows>
    3. Dual Boot : Install Both OS on same system <Need to choose OS at startup>
